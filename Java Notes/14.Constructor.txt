Constructor:
It is a Special method, When the object is created the Constructor will invoke automatically.

Rule:
1.Class name and constructor name should be same
2.Constructor should not have return type.

Non-Parameterized

Constructor chaining:
To reduce the number of object creation
The process of calling one constructor from another with respect to current object creation is called constructor chaining

super();
