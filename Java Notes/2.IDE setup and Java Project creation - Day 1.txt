Java config and setup
------------------------
Go to google and download JDK (1.8)
Install JDK
Verify JDK (cmd ---> java -version  --? Press enter key

Download Eclips - (Oxygen 3a)



Step 1: (Project creation):
Create JAVA Project  (Files----> New---> Click Java Project ---> Give Project name(Must follow Pascal notation) -----> click Finish

Step 2: (Package creation)
Inside the project ---> To create package---> (Right click src folder of ur project --> Click new --> package --> then give package name --> Finish
***Everything must be small letter**
Package name should be start with org. or com.

step 3: (Class Creation)
Inside your package  ---> (Right click your package ---> click new ---> click class ----> Give class name ----> (Must follo pascal notation ---> Finish


step 4: - Method creation
Inside a class ---> Give method name (Following camel notation) and press control + space 

system.out.println() ------> sysout ctrl+space


Main method - Execution start from Main method


Object creation ---->  ClassName ObjName = new ClassName();
